# VetFleet Sniffer GitHub Pages Package

This folder is the clean public-repo package for the mobile landing page.

Files included:
- `index.html`
- `logo.png`
- `Leak Check Flyer (7).pdf`
- `thanks/index.html`
- `.nojekyll`

## Publish Steps

1. Create a new public GitHub repo named `vetfleet-sniffer-page`.
2. Upload everything in this folder to the root of that repo.
3. Open `index.html` and replace:

`https://YOUR-GITHUB-USERNAME.github.io/vetfleet-sniffer-page/thanks/`

with your actual Pages URL.

Example:

`https://aztectech82-cell.github.io/vetfleet-sniffer-page/thanks/`

4. Commit the change.
5. In GitHub repo settings, open `Pages`.
6. Set source to:
   - Branch: `main`
   - Folder: `/ (root)`
7. Save.

Your site URL will look like:

`https://YOUR-GITHUB-USERNAME.github.io/vetfleet-sniffer-page/`
